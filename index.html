Threads

Threads is a mock ecommerce shop selling a variety of clothing and accessories. It is being developed with the MERN stack. The products are stored using MongoDB populated by the Fake Store API. I am collaborating on this project via GitHub with users pvquinn and CrankyMandaPanda. To ensure a smooth development process, we are using Git Flow with at least one person approving each pull request. The full repo is available below.
